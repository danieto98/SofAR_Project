void StateMachine::callback_command ( const labeled_slam::Command::ConstPtr &  msg)

Callback function for text_command subscriber.

Callback-function interpreting command from command-recognition node.

Allowed commands are "drive", "go to", "label", "listen" commands go_to and label are using an argument. For the other commands, the argument is just ignored. According function of the state_member are called on each command Depending on the true object-type in the state_-variable, the function of one specific state is called (POLYMORPHISM!)
